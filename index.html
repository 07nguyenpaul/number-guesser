Number Guesser

    Guess What Number I'm Thinking Of...

    Instruction on how to play the game:

  1. 1. Pick a number between 1-100 and press GUESS
  2. 2. OR set a MIN and MAX range you would like play at.
  3. 3. Keep guessing, don't give up.
  4. 4. If you get it correct, it gets a little harder. Try it again but the MIN and MAX range will increase and decrease by 10. GOOD LUCK!
  5. 5. Reset the Game and Start over.
